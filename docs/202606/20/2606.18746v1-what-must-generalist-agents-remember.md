## 摘要
本文对通用型智能体在多个环境和目标中近乎最优地行动时必须在记忆中存储什么内容进行了形式化描述。研究表明，当两个领域共享一个观察瓶颈但需要不相容的最优行动时，任何近乎最优的策略都会在该瓶颈处诱导出不同的记忆分布。这一结果推导出一个分离定理：足够成功的智能体不能仅依赖当前状态观察，而必须在记忆中保留与领域相关的信息。本文进一步证明，如果智能体的记忆包含足够的信息来估计相关目标的价值，那么该记忆可用于近似重建智能体的局部转移动力学。综合这些结果，记忆被刻画为支持领域消歧、转移模型重建以及通用型智能体规划的基底。

## Abstract
This paper develops a formal account of what generalist agents must store in memory in order to act near-optimally across multiple environments and goals. It shows that when two domains share an observational bottleneck but require incompatible optimal actions, any uniformly near-optimal policy must induce distinct memory distributions at that bottleneck. The result yields a separation theorem: sufficiently successful agents cannot rely only on current state observations, but must preserve domain-relevant information in memory. The paper further shows that if an agent's memory contains enough information to estimate values for related goals, then that memory can be used to approximately reconstruct the agent's local transition dynamics. Together, these results characterize memory as the substrate that supports domain disambiguation, transition-model reconstruction, and planning for generalist agents.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 2.1 形式化框架

- **设置**：有限状态空间 S、动作空间 A、折扣因子 γ。每个领域 σ∈Σ 有初始分布 μ_σ 和转移核 P_σ(·|s,a)。智能体观察历史 h_t，通过记忆函数 M_t = f(h_t) 压缩历史，策略通过共享策略头 π_M(a|M_t, ψ) 生成动作（ψ 为目标）。
- **关键概念**：
  - **控制等价**：若两个领域在所有状态和目标下的近优动作集相同，则视为控制等价。
  - **共享瓶颈与不相容动作集**（Assumption 1）：存在一个状态 s†（瓶颈），两个领域在该状态对于相同目标需要不同的动作集 G_σ 和 G_σ̃，且由值间隙 ε_gap 分离；近优策略必须高概率（≥ ρ）到达 s†。
  - **辅助探针目标**（Definition 2）：一类测试函数 F 被 Ψ_F 生成，使得最优动作-价值等于测试函数在转移核下的期望。
  - **价值充分记忆**（Definition 3）：记忆状态 M_t 能支持一个值头 q_M 逼近所有辅助目标在各动作下的最优值，误差 ≤ ε_val。

### 2.2 主要理论结果

- **定理1（记忆分离）**：若智能体在所有领域和目标上统一满足 ε_opt 近优性（ε_opt < ε_0 = ½ ργ^T ε_gap），则对于每对满足 Assumption 1 的领域，在瓶颈状态 s† 处，两个领域的记忆条件分布的总变差距离 ≥ [1 - η_σ - η_σ̃]_+ > 0。即记忆必须分离领域。
- **定理2（解码器存在性）**：若记忆是 ε_val-价值充分的，且辅助目标集能 ε_span-生成测试类 F，则：
  1. 相同记忆状态暗示局部转移核在度量 d_F 下接近（误差 ≤ 2(ε_val+ε_span)）。
  2. 存在一个解码器 D_P: M × A → P(S)，使得解码核与真实核在可达历史上的 d_F 距离 ≤ 2(ε_val+ε_span)。
  3. 若满足规划稳定性（Assumption 3），解码核可用于近似规划，规划误差 ≤ S(2(ε_val+ε_span))。

### 2.3 关键技术细节

- **证明思路**：定理1通过构造比较策略，利用价值差距和瓶颈到达概率将近优误差转化为动作选择错误概率，进而推出记忆分布必须不同。定理2利用辅助目标作为“测试函数”，通过值头的逼近性质将记忆状态与转移核的期望链接起来，进而构造出解码器。
- **未使用复杂网络结构**：记忆函数 f 可以是任意可学习映射（如 RNN 的隐藏状态），理论不假设贝叶斯更新或显式上下文变量。

## 3. 实验设计

### 3.1 环境与基准

- **环境**：ForkWorld，一个 T 型网格世界（4×7 网格）。走廊长度 c=4，臂长 ℓ=3。起始 s0=(0,3)，瓶颈 s†=(3,3)，上端目标 G_up=(3,6)，下端目标 G_down=(3,0)。动作集：{up, down, left, right, stay}。
- **领域隐藏变量**：σ∈{normal, swapped}，改变 up/down 动作的垂直映射。normal 时 up 向上、down 向下；swapped 时相反。
- **目标 ψ**：{up, down}，指定到达哪个端点。奖励：正确目标 +1，错误或超时 0，每步成本 0.01。目标在观测中给出，但领域 σ 不给出。
- **观测**：记忆型智能体观测 [x, y, ψ_id, 1]；Oracle 额外给出 σ_id。

### 3.2 对比方法

| 方法 | 说明 |
|------|------|
| Memoryless DQN | 两层 MLP，仅当前观测，无记忆 |
| Stacked DQN (k=3, +actions) | MLP + 3步历史窗口 + 2步动作 |
| DRQN (states) | GRU 隐状态，仅观测序列 |
| DRQN (states+actions) | GRU + 上一步动作 |
| Oracle DQN | 内存less + 已知 σ_id，性能上限 |
| Oracle DRQN | GRU + 已知 σ_id，另一上限 |

### 3.3 评估设置

- **训练**：8个随机种子，5000 episodes。ϵ-greedy 从1.0线性衰减至0.05（3000 episodes），γ=0.99，Adam lr=1e-3。
- **评估**：每50 episodes 一次贪婪评估，每个条件5个episode（4个条件共20）。最终结果报告8种子均值±标准误。

- **解码器**：训练两层 MLP 从 (M_t, a_t) 预测 s_{t+1}，采用 80/20 划分，报告错误率。

## 6. 论文的主要结论与发现

1. **记忆是必要条件**：在 ForkWorld 中，无记忆 DQN 仅达到19%成功率（低于随机50%），因梯度冲突导致无法学习稳定策略。带记忆的 agent 达到75-88%，接近 Oracle 的100%。
2. **记忆在瓶颈处分离领域**：首次访问瓶颈时所有 agent 无法区分领域（准确率≈50%）；后续访问后，DRQN 和 Stacked DQN 的记忆表示达到97-100% 的领域分类准确率，验证了定理1。

4. **记忆是领域消歧、模型重建和规划的基底**。
