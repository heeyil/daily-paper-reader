## 摘要
测试时缩放提升了语言模型的推理能力，但现有方法常面临艰难权衡：长思维链采样仍是单线程的，而句子级或解级搜索可能计算成本高昂且难以端到端训练。我们提出局部分支路由（LBR），一种基于令牌的测试时缩放框架，扩展小型局部前瞻树，通过语言模型前向传播所有采样分支，并使用轻量级路由器选择要提交的深度1子树。通过在候选局部未来的隐藏状态上进行路由，LBR允许每个令牌决策利用根部下一个令牌分布之外的证据，同时避免完整的解级搜索。由此产生的剪枝-移动-增长解码过程保留了离散分支标识，并定义了可处理的树轨迹似然：新生长节点在首次采样时被计数，路由器决策被赋予显式概率。这使得能够通过可验证奖励进行端到端强化学习，在相同的似然比原则下联合优化基础模型和路由器，如同离散令牌RLVR。在合成层次规划任务中，LBR显示候选隐藏状态提供了有用的路由证据。在数学推理基准测试中，LBR在Pass@1和Pass@32上均优于离散思维链、普通离散令牌RLVR以及RL兼容的软令牌分支基线。这些结果表明，轻量级局部分支为语言模型测试时缩放提供了一种高效、可训练且离散的形式。

## Abstract
Test-time scaling improves language-model reasoning, but existing approaches often face a difficult trade-off: long chain-of-thought sampling remains single-threaded, while sentence- or solution-level search can be computationally expensive and hard to train end-to-end. We introduce Local Branch Routing (LBR), a token-level test-time scaling framework that expands a small local lookahead tree, forwards all sampled branches through the language model, and uses a lightweight router to select the depth-1 subtree to commit. By routing over the hidden states of candidate local futures, LBR allows each token decision to use evidence beyond the root next-token distribution while avoiding full solution-level search. The resulting prune-shift-grow decoding process preserves discrete branch identities and defines a tractable tree-trajectory likelihood: newly grown nodes are counted when first sampled, and router decisions are assigned explicit probabilities. This enables end-to-end reinforcement learning with verifiable rewards, jointly optimizing the base model and router under the same likelihood-ratio principle as discrete-token RLVR. On synthetic hierarchical-planning tasks, LBR shows that post-candidate hidden states provide useful routing evidence. On mathematical reasoning benchmarks, LBR improves both Pass@1 and Pass@32 over discrete chain-of-thought, vanilla discrete-token RLVR, and RL-compatible soft-token branching baselines. These results suggest that lightweight local branching offers an efficient, trainable, and discrete form of language-model test-time scaling.

- **问题**：现有语言模型测试时缩放方法在效率与可训练性之间存在两难困境。

- **背景**：token 级别的局部选择（如数字、运算符、变量、短语）可决定模型进入的下一潜在状态，但标准自回归解码在观察附近候选隐藏状态前已提交单一 token。
- **核心目标**：提出一种轻量级、可训练的 token 级别测试时缩放方法，利用局部前瞻树的路由来利用候选隐藏状态，同时避免全局搜索。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：在每个解码步骤，LBR 构建一个深度 L、宽度 K 的局部前瞻树，前向传播所有采样分支，然后使用轻量路由器选择深度 1 子树提交，剪除其他分支，将选中子树移位并生长一层。
- **关键技术细节**：
  - **生长**：从当前前缀开始，为每个活动节点从模型分布 ˜πθ 采样 K 个子节点，深度 L。所有节点均通过语言模型前向传播，存储隐藏状态 h_t(v)。
  - **路由**：路由器 ρ_φ 基于整个树的隐藏状态，通过交叉子树集合注意力机制为每个深度1候选子树打分，并采样选择提交的子树。路由器输出分布为 softmax 归一化后的分数。
  - **剪枝与移位-生长**：丢弃未选中子树，将选中子树的根 token 加入到已提交前缀；其剩余子节点作为下一解码步的部分前瞻树，并生长一层新节点恢复深度 L。
  - **轨迹似然**：对数概率 = 所有新生长节点 token 对数概率之和 + 所有路由器决策对数概率之和。该似然可用于 RLVR 训练的似然比目标。
- **公式/算法流程（文字说明）**：
  - 初始化：构建首个深度 L 树。
  - 循环每个解码步：
    1. 生长当前树的活动节点（采样 K 个子节点并前向传播）。
    2. 路由器观察完整树，输出分布并采样选择深度1子树 k*。
    3. 提交 token = 该子树的根 token。
    4. 剪除其他子树，选中子树移位。生长一层新节点恢复深度 L。

## 6. 主要结论与发现

1. **LBR 在数学推理上显著优于所有基线**：在 1.5B 和 7B 模型上，LBR (L=1) 在 6 个基准中 Pass@1 和 Pass@32 均超过离散 CoT、vanilla RLVR 和软 token 分支方法。增加深度 (L=2) 进一步改进，尤其 7B 模型。
2. **后候选隐藏状态包含关键决策信息**：合成任务中，LBR 利用分支后隐藏状态的路由准确率高于依赖预分支状态的离散 CoT；软 token 合并会模糊概念身份。
3. **交叉子树对比注意力至关重要**：消融实验显示，移除交叉注意力的独立路由器显著低于完整路由器，证明对比性评估的必要性。
4. **训练中路由器与基模型共同适应**：路由器熵降低，同时基模型在路由位置的熵也降低，表明两者共同学习使候选分支更易区分。

## 7. 优点

- **方法创新**：在 token 级别引入局部前瞻树路由，利用隐藏状态作为决策证据，保持离散分支标识，避免软 token 合并导致的信息模糊。
- **可端到端训练**：定义可微树轨迹似然，支持 RLVR 训练，统一优化基模型和路由器。
- **高效性**：解码使用滚动局部树，避免全局搜索，计算开销可控；路由器为轻量级（内部维度128，4头注意力）。
- **实验设计严谨**：合成任务提供了机制验证（隐藏状态探测、概念身份探测），数学推理覆盖广泛数据集，并进行消融和动态分析。
- **可解释性**：通过探针实验揭示 LBR 相对于离散 CoT 和软 token 的优势来源。

## 8. 不足与局限

- **局部性**：LBR 仅利用浅层局部前瞻信息（深度 L），无法进行完全的长程规划或全局搜索。路由信号受限于局部窗口。
- **计算开销随深度增长**：增大深度 L 可提供更丰富的路由信号，但会显著增加前向传播成本（宽度 K 也呈指数增加），存在效率-效果权衡。
- **未探索自适应策略**：未结合自适应深度、选择性全局搜索或学习何时需要更长规划，限制了在复杂任务上的扩展性。
- **实验局限性**：仅评估了特定模型系列（DeepSeek-R1-Distill-Qwen）和数学推理/合成规划任务，通用性需更多验证。未在代码、常识推理等领域测试。
- **潜在偏差**：训练集 DeepScaleR-Preview-Dataset 可能对评估基准有潜在知识泄露或分布匹配风险。
- **可重复性**：路由器架构细节和超参数在附录给出，但完整训练代码和超参数搜索过程未提供。
