## 摘要
零样本物体目标导航要求具身体在没有任何预先训练的情况下探索并定位目标物体。为此，近期的方法利用了基础模型，但它们通常依赖于静态先验且缺乏适应性，导致重复错误和昂贵的试错。本文提出了一种自演进的零样本物体目标导航框架，能够在测试时持续改进。具体而言，我们通过从过去轨迹中提取可操作知识来构建智能体规则记忆。然后，我们提出了一种基于上置信界（UCB）的检索策略，通过平衡语义相关性和历史成功率来选择有效规则。此外，我们引入了一个记忆引导的预反思模块，该模块在动作执行前预测潜在结果，从而减少低效探索。大量实验表明，我们的方法优于现有的零样本基线方法，在减少不必要步骤的同时实现了10.1%的成功率提升。

## Abstract
Zero-Shot Object-Goal Navigation (ZS-OGN) requires embodied agents to explore and locate target objects without any prior training. To this end, recent methods leverage foundation models. But they typically rely on static priors and lack adaptation, which leads to repeated errors and costly trial and error. In this paper, we propose a self-evolving ZS-OGN framework that enables continuous test-time improvement. Specifically, we build an agentic rule memory by extracting actionable knowledge from past trajectories. Then, we propose a retrieval strategy based on upper confidence bound, selecting effective rules by balancing semantic relevance and historical success. In addition, we introduce a memory-guided preflection module that forecasts potential outcomes before action, reducing inefficient exploration. Extensive experiments show that our method outperforms existing zero-shot baselines, achieving a 10.1\% improvement in success rate with fewer unnecessary steps.

- **研究问题**：零样本物体目标导航（ZS-OGN）要求智能体在没有预先训练的情况下，在未知环境中探索并定位目标物体。现有方法依赖基础模型（如LLM/VLM）提供的静态先验知识，无法根据环境实时反馈进行自适应调整，导致重复错误和高成本试错。
- **核心挑战**：如何在有限测试步数预算下，同时实现**在线适应**和**行动预判**，避免被动修正和低效探索。
- **背景意义**：ZS-OGN是具身人工智能的基础任务，现有训练型方法需要大量数据和模拟，泛化性差；训练免费方法虽零样本但缺乏进化能力。本文旨在将导航转化为持续自我改进的过程。

## 2. 方法论：核心思想、关键技术细节
### 2.1 整体框架（EvolveNav）
- 两大核心组件：**帧内-预反思探索**（Intra-Episode Preflective Exploration）和**帧间-规则自演进**（Inter-Episode Rule Self-Evolution）。
- 智能体在测试时不断从自身经验中提取可操作规则，并利用规则预判风险，从而实现持续自改进。

### 2.2 空间映射与底层控制
- 基于RGB-D输入和里程计，动态构建2D占据网格地图，提取候选前沿点（frontier）。
- 底层导航控制器执行到目标点，最大步数为H，避免死锁。

### 2.3 自演进智能体规则记忆（Self-evolving Agentic Rule Memory）
- **规则生成**：每个导航回合（无论成功或失败）后，使用LLM分析轨迹，识别关键决策步骤，利用**语义驱动信用分配**（Semantic-driven Credit Assignment）为每个步骤分配初始信用分数。
  - 步骤i的语义相似度分数：\( S_i = \alpha \cdot S_{room}^{(i)} + (1-\alpha) \cdot S_{obj}^{(i)} \)（式1）
  - 信用分数：\( C_i = S_i - S_{i-1} \)（式2）
  - 新规则r的初始支持分数：\( S_r = \sum_i w_i \cdot C_i \)（加权和，式3）
- **动态管理**：采用UCB（置信上界）策略平衡规则利用与探索。
  - 预期效用动量更新：\( \mu_r \leftarrow \eta \cdot \mu_r + (1-\eta) \cdot S_r \)（式4）
  - UCB分数：\( \text{UCB}(r) = \mu_r + \beta \sqrt{ \frac{\ln T}{N_r} } \)（式5）
  - 新规则（N_r=0）初始UCB设为∞，确保被探索。
- 每次新回合前，从规则库中选择Top-K高UCB规则注入LLM推理上下文。

### 2.4 记忆引导预反思模块（Memory-guided Preflection）
- **主动风险规避**：在执行动作前，对于每个候选前沿点，智能体检索相关规则和历史失败案例，让LLM进行风险预测（如“可能进入死胡同”、“视觉线索冲突”）。
- **替换直接选择**：LLM必须执行显式推理，预测每个前沿点的失败概率，再选择最优、风险最小的路径。
- 核心优势：将后验修正转为先验预防，显著减少物理试错代价。
