## 摘要
VLA模型已成为将网络规模数据的语义知识迁移到物理机器人控制中的强大范式。然而，当前的单帧架构存在固有局限性：忽视历史动态的时间近视、高级指令与低级电机命令之间的推理鸿沟，以及自回归标量解码导致的推理低效。本文提出MIRTH，一个旨在应对这些挑战的统一框架。MIRTH通过三项关键创新增强预训练的VLA骨干网络：(1)双尺度时空记忆枢纽，将长期场景演变和短期运动趋势压缩为紧凑嵌入；(2)通过互信息目标优化的潜在推理标记，开辟语义规划空间以对齐多模态上下文与动作轨迹；(3)并行动作解码方案，用向量级预测替代自回归生成以最大化控制吞吐量。在LIBERO仿真基准和真实世界LeRobot平台上的广泛评估表明，MIRTH实现了最先进的性能，并展现出突现的错误恢复能力。代码和收集的数据集已发布在 http://github.com/kiva12138/mirth。

## Abstract
VLA models have emerged as a powerful paradigm for transferring semantic knowledge from web-scale data to physical robotic control. However, current single-frame architectures suffer from intrinsic limitations: temporal myopia that discards historical dynamics, reasoning gaps between high-level instructions and low-level motor commands, and inference inefficiency due to autoregressive scalar decoding. In this work, we propose MIRTH, a unified framework designed to address these challenges. MIRTH augments a pretrained VLA backbone with three key innovations: (1) dual-scale temporal memory hubs that compress long-term scene evolution and short-term motion trends into compact embeddings; (2) latent reasoning tokens optimized via a mutual-information objective carving out a semantic plan space to align multimodal context with action trajectories; and (3) a parallel action decoding scheme that replaces autoregressive generation with vector-wise prediction to maximize control throughput. Extensive evaluations on the LIBERO simulation benchmark and a real-world LeRobot platform demonstrate that MIRTH achieves state-of-the-art performance and exhibiting emergent error recovery capabilities. The codes and collected datasets are released at http://github.com/kiva12138/mirth.

## 1. 核心问题与整体含义（研究动机与背景）

当前 Vision-Language-Action (VLA) 模型（如 RT-2、OpenVLA）依赖单帧输入，存在三大固有局限：
- **时间短视（temporal myopia）**：丢弃历史动态信息，物体被遮挡后丢失跟踪，动作易漂移。
- **推理鸿沟（reasoning gap）**：高层语言指令与底层电机命令之间缺乏中间规划，要么缺乏可解释性，要么需要昂贵的手工文本监督。
- **推理效率低下**：将连续动作量化为离散 token 后采用自回归解码，每个动作维度需生成一个 token，导致控制频率低、实时性差。

MIRTH 旨在统一解决上述问题，引入时序记忆、潜在推理和并行动作解码，使 VLA 模型在保持语义丰富性的同时具备历史感知、高效且鲁棒的物理控制能力。

## 2. 方法论：核心思想、关键技术细节

### 2.1 双尺度时间记忆枢纽（Dual-scale Temporal Memory Hubs）
- **工作空间记忆（长期）**：使用 K 个指数移动平均（EMA）以不同衰减率保存场景布局，并融合一阶/二阶速度统计量，得到固定大小的长期特征图 \(M_t^{\text{work}}\)。
- **短地平线记忆（短期）**：维护最近 w 帧的队列，通过含衰退偏置的 temporal attention 加权求和，得到 \(M_t^{\text{short}}\)。
- **融合**：用 sigmoid 门控自适应融合两个记忆，再以 prefix 或 infusion 方式注入 VLM 骨干。prefix 方式将记忆展平后前置，性能更好；infusion 方式保持序列长度恒定，效率更高。

### 2.2 潜在推理令牌（Latent Reasoning Tokens）与互信息最大化
- 在输入序列中插入可学习推理令牌 \(R_t \in \mathbb{R}^{m \times D}\)，位于条件（视觉+语言）与动作令牌之间。
- 从 VLM 输出中提取推理表示 \(z^R\)、动作表示 \(z^A\)、上下文表示 \(z^X\)。
- 优化 InfoNCE 对比损失：
  - \(L_{ra}\)：最大化推理表示与动作表示之间的互信息；
  - \(L_{rx}\)：最大化推理表示与上下文表示之间的互信息；
  - 总损失 \(L_{\text{MI}} = \lambda_{ra} L_{ra} + \lambda_{rx} L_{rx}\)。
- 此设计让推理令牌自动学习语义规划空间，无需人工文本标注。

### 2.3 并行动作解码（Parallel Action Decoding）
- 每个动作向量（含 \(N_F\) 维度）对应一个 token，替代标量级 token化。
- 对动作 token 的隐藏状态采用并行投影（全局拼接后投影），一次性输出整个动作轨迹，避免自回归。
- 使用 \(\ell_1\) 回归损失 \(L_{\ell_1}\) 训练。
- 最终损失：\(L = L_{\ell_1} + \lambda_{\text{MI}} L_{\text{MI}}\)。

## 3. 实验设计

### 数据集/场景与 Benchmark
- **仿真**：LIBERO 基准套件，包含 4 个套件（Spatial、Object、Goal、Long），共 40 个任务。每个任务评估 30 次随机初始化。
- **真实世界**：LeRobot 开源平台，包含 5 个任务组（Basic Tasks、Mechanism Operations、Scene Rearrange、Category Reasoning、Semantic Recipe），共 126 个独特任务，收集 1000 条专家轨迹，每个任务 50 条。评估时每个任务 30 次试验。

### 对比方法
- **LibERo 仿真**：Diffusion Policy、Octo、OpenVLA、OpenVLA-OFT。
- **LeRobot 真实**：同前，额外包含 MIRTH 的消融变体。

### 主要实验结果
- **LIBERO**：MIRTH 在四个套件上平均成功率 98.1%，大幅超越 OpenVLA（76.5%）和 OpenVLA-OFT（97.1%）。尤其在 LONG 套件上达 95.3%（OpenVLA 仅 53.7%）。
- **LeRobot**：MIRTH 在所有任务组均最高，推理吞吐量接近甚至超过轻量基线。

### 消融实验（LIBERO-Long）
- 移除工作空间记忆 → 下降 1.3%；移除短期记忆 → 下降 0.9%；移除两者 → 下降 2.1%。
- 移除 MI 损失 → 下降 0.8%；移除推理令牌 → 下降 1.4%。
- 验证了两类记忆枢纽和互信息推理的必要性。

### 分析实验
- **线性探测**：MIRTH 的状态/速度估计 MAE 远低于 OpenVLA，证明记忆成功捕捉运动动态。
- **帧乱序测试**：打乱短期记忆顺序 → 成功率从 95.3% 降至 88.0%，说明模型依赖因果时序。
- **推理令牌 t-SNE**：无监督形成任务语义聚类（如开启容器类、拾放类）。
- **错误恢复**：MIRTH 恢复率 12.1%，远超 OpenVLA（5.2%）和消融模型（8.7%）。
- **记忆集成策略**：Prefix 策略（95.3% / 64.4 Hz）优于 Infusion（93.1% / 70.0 Hz），但 Infusion 吞吐更高。
- **解码范式对比**：全局拼接向量解码收敛最快，最终采用此法。
- **动作块大小**：块大小 10 为最佳均衡（成功率 94.0%，吞吐 62 Hz）。

## 6. 主要结论与发现
1. MIRTH 通过双尺度记忆枢纽有效克服单帧模型的时间短视，在长程依赖任务中提升幅度显著（LIBERO-Long 提高 41.6%）。
2. 互信息驱动的潜在推理令牌无需文本监督即可自发形成语义规划空间，并赋予模型错误恢复能力。
3. 并行动作解码在不牺牲性能的前提下大幅提升控制吞吐量，有利于实时部署。
4. 各组件（长期记忆、短期记忆、推理令牌、互信息约束）均对最终性能有贡献，缺一不可。

## 7. 优点
- **创新性**：首次将互信息对比学习用于 VLA 的潜在推理，并设计双尺度时间记忆以固定 token 数压缩历史。
- **效率**：并行动作解码将因果延迟从标量级降至向量级，吞吐接近实时需求。
- **可解释性分析**：t-SNE 可视化、线性探测、帧乱序等实验深入揭示了模型工作机制。
- **开源贡献**：代码、数据集已公开，便于复现与扩展。
- **鲁棒性**：在仿真和真实平台上均达 SOTA，并展现出意外规划/恢复能力。

## 8. 不足与局限
- **可解释性不足**：潜在推理令牌虽有效，但不可读，难以像 Chain-of-Thought 那样审计决策过程。
- **实验范围有限**：仅验证了固定单臂操作，未涉及双臂协同、移动操作或动态人机交互。
- **长程记忆瓶颈**：压缩记忆固定大小，极端长程任务（> 数百步）仍可能遗忘。

- **硬件依赖**：训练需高端 GPU（RTX Pro 6000），可能限制可复现性。
