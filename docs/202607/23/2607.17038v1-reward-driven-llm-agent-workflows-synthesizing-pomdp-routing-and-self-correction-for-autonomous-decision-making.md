## 摘要
本文通过设计和优化智能体工作流，解决了当前大型语言模型（LLM）智能体应用中的关键技术挑战，包括长程规划、稀疏奖励归因和动态环境交互。所提出的架构基于核心人工智能范式的综合：视觉、语言、生成、图、多模态、强化和智能体智能。与依赖静态提示且缺乏稳健感知-行动循环的传统基线模型不同，我们的方法引入了一种部分可观测马尔可夫决策过程（POMDP）路由机制。该机制通过一个内部的自校正奖励模型进行增强，该模型在执行前评估决策轨迹。通过整合多模态输入和先进的强化学习原则（如近端策略优化和值函数近似），智能体保持长期结构记忆，并动态调整其推理路径以减少误差累积。在ALFWorld具身仿真环境和WebShop在线导航基准上的实验表明，任务成功率和轨迹效率相比主流基线（如标准ReAct框架）绝对提高了24.5%。全面的消融研究证实了奖励驱动批判模块在抑制幻觉率方面的重要贡献。本研究将强化学习和基于图的记忆的理论基础与自主智能体工作流相连接。最终，所提出的架构为开发复杂多步自主系统中的人工智能技术提供了一个实用、可扩展的参考框架。代码可在 https://github.com/01Amez/RLAW_Implementation 获取。

## Abstract
This paper addresses key technical challenges in current large language model (LLM) agent applications, including long-horizon planning, sparse reward attribution, and dynamic environmental interaction, by designing and optimizing an intelligent agent workflow. The proposed architecture is based on the synthesis of core AI paradigms: Visual, Language, Generative, Graph, Multimodal, Reinforcement, and Agent Intelligence. Unlike conventional baseline models that rely on static prompting and lack robust perception-action loops, our approach introduces a Partially Observable Markov Decision Process (POMDP) routing mechanism. This mechanism is augmented with an internal, self-correcting reward model that evaluates decision trajectories before execution. By integrating multimodal inputs and advanced reinforcement learning principles (such as proximal policy optimization and value function approximation), the agent maintains long-term structural memory and dynamically adapts its reasoning pathways to mitigate error accumulation. Empirical experiments on the ALFWorld embodied simulation environment and the WebShop online navigation benchmark demonstrate a 24.5% absolute improvement in task success rate and trajectory efficiency over mainstream baselines like the standard ReAct framework. Comprehensive ablation studies confirm the significant contribution of the reward-driven critique module in suppressing hallucination rates. This research bridges theoretical foundations of reinforcement learning and graph-based memory with autonomous agent workflows. Ultimately, the resulting architecture offers a practical, scalable reference framework for developing artificial intelligence technologies in complex, multi-step autonomous systems. Code is available at https://github.com/01Amez/RLAW_Implementation.

## 1. 核心问题与整体含义（研究动机与背景）

当前大型语言模型（LLM）驱动的自主智能体在复杂动态环境中面临三大关键挑战：

- **长程规划退化**：由于自回归生成缺乏全局目标导向，早期推理错误会逐步级联放大（“级联幻觉”），导致整个执行路径失效。
- **稀疏奖励归因**：传统LLM仅基于最大似然估计生成动作，缺乏显式的决策质量评估信号，在只有最终成功/失败反馈的任务中难以判断中间步骤的效用。
- **有限的结构记忆**：标准LLM依赖固定长度上下文窗口，难以维持对部分可观测环境的精确世界状态表示（如房间内物体摆放、抽屉开关状态），容易重复执行无效动作或陷入死循环。

为解决这些问题，论文提出将LLM智能体形式化为**部分可观测马尔可夫决策过程（POMDP）**，并引入**内部奖励驱动的自校正机制**，将语言生成与强化学习的数学优化目标融合，使智能体具备“提出-批判-执行”的闭环推理能力。

## 2. 方法论（核心思想与关键技术细节）

### 2.1 整体架构：RLAW（Reward-Driven LLM Agent Workflow）

架构包含三个阶段，形成“Propose–Critique–Execute”循环：

- **Actor阶段（LLM策略）**：根据当前观测 \(o_t\)、历史轨迹 \(H_t\) 以及**图记忆模块 \(M_t\)**（由Graph Attention Network维护的场景图），生成候选推理链（Thought）和执行动作（Action）。动作以JSON格式输出，约束在有限动作空间中。
- **Critique阶段（内部奖励模型）**：轻量级Critic模型 \(f_\phi\) 接收Actor生成的候选动作，输出连续得分 \(R_{\text{critique}} \in [0,1]\)，包含逻辑一致性、安全性、目标对齐度等指标。若得分低于阈值 \(\tau=0.7\)，则拒绝该动作，并生成自然语言反馈指导Actor重生成（最多允许3次重试）。
- **执行阶段**：通过批判的动作在真实环境中执行，获得环境稀疏奖励 \(R_{\text{env}}\) 和下一观测 \(o_{t+1}\)。最终优化目标为加权和：\(J(\pi_\theta) = \mathbb{E}_\tau \left[ \sum_t \gamma^t (R_{\text{env}} + \lambda R_{\text{critique}}) \right]\)，其中 \(\lambda=0.5\) 平衡内外奖励。

### 2.2 关键技术细节

- **POMDP形式化**：用7元组 \(\langle S, A, T, R, \Omega, O, \gamma \rangle\) 建模部分可观测性，基于历史信念状态 \(b_t\) 更新。
- **图记忆（Graph Intelligence）**：使用Graph Attention Network（GAT）动态更新节点特征，将实体（物体、房间）及其关系（空间、功能）编码为语义场景图，作为LLM上下文中的序列化文本输入。
- **Critic训练**：基于LLaMA-3-8B-Instruct，使用LoRA微调（rank=16, \(\alpha=32\)），在50,000条推理与批判轨迹上训练3个epoch，优化因果交叉熵损失，输出符合JSON schema的评分。
- **PPO策略优化**：形式上使用PPO的裁剪目标稳定策略更新，但实际Actor（LLM）保持冻结，仅Critic通过监督学习获得奖励预测能力。

## 6. 主要结论与发现

1. **RLAW显著提升成功率**：ALFWorld上78.6%（ReAct 54.1%），WebShop上65.8%（ReAct 42.3%），绝对提升24.5%。
2. **幻觉率大幅降低**：ALFWorld从28.5%（ReAct）降至12.4%，WebShop从31.2%降至14.1%，证明Critic模块有效过滤无效动作。
3. **参数效率优越**：8B参数的RLAW（78.6%）超越70B参数的Standard ReAct（75%），表明架构设计比单纯增大模型更关键。
4. **消融证实Critic为核心**：去掉Critic后成功率下降17.4%（从78.6%→61.2%），平均步数增加2.1步，说明内部奖励机制是性能提升的主因。
5. **图记忆模块辅助**：去掉后成功率下降10.2%（78.6%→68.4%），但Critic依然是更重要的组件。

## 7. 优点

- **理论创新**：将LLM智能体严格形式化为POMDP，并通过内部Critic近似值函数，将语言生成纳入强化学习优化框架，克服了“生成即行动”的盲目性。
- **系统设计清晰**：Actor-Critic分离、三重循环（提出-批判-执行）、结构图记忆，每个模块职责明确，易于扩展和调试。
- **实用性强**：在消费级GPU上（单卡4090）可实现高效推理，量化后8B模型即可超越大模型，工业部署门槛低。
- **开源可复现**：提供完整代码实现。

## 8. 不足与局限

- **推理延迟增加**：每步2.15秒（vs ReAct 1.20s），不适合实时或低延迟任务（如机器人控制、在线对话）。
- **上下文窗口瓶颈**：即使有图记忆压缩，当轨迹超100步时仍出现“lost in the middle”问题，长序列下性能可能退化。
- **Critic质量依赖**：若Critic训练数据有限或存在偏差，可能产生误判（拒绝正确动作或接受错误动作），导致无限重试循环。
- **多模态整合有限**：当前仅将视觉特征简单序列化，未使用端到端视觉-语言模型（如ViT），视觉信息利用率不足。
- **实验范围**：仅在两个模拟环境测试，未在真实物理机器人或复杂开放域（如网页填写、代码执行）中验证，泛化性有待评估。未报告多次运行的标准差，统计显著性分析不完整。
