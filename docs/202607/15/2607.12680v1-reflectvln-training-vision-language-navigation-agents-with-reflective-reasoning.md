## 摘要
现有视觉-语言导航方法通常将视觉-语言模型与航路点解码器结合以生成多步动作计划，但它们通常缺乏明确的闭环机制来跟踪语义进展、诊断执行失败，并在长时域导航中从错误累积中恢复。为解决这一问题，我们提出ReflectVLN，一种智能体化的VLN框架，通过双向交互的意图智能体和执行智能体组织决策。意图智能体执行子任务分解与反思，生成可执行的子任务描述作为纠正计划。在这些描述的约束下，执行智能体将其落地为当前观察下的短时域动作，同时监控子目标进展并检测偏离轨道行为。关键在于，ReflectVLN实现了闭环双向通信：执行智能体发出进度和偏差信号，按需触发反思和子任务更新；意图智能体返回结构化指导，重新调整后续动作以进行恢复。为了鼓励时序一致的决策并带有可解释的中间推理，我们引入动作思维链（Action-CoT），一种路径条件双查询训练方案用于动作生成。在标准VLN基准上的实验表明，ReflectVLN在有限的数据预算下提高了成功率和路径效率，同时在推理时具有较低的训练成本和更少的高级意图调用，并提供可解释的中间决策以用于分析和协作。代码地址：https://github.com/AIprogrammer/ReflectVLN

## Abstract
Existing vision-language navigation methods often couple a VLM with waypoint decoders to produce multi-step action plans, but they typically lack an explicit closed-loop mechanism for tracking semantic progress, diagnosing execution failures, and recovering from error accumulation in long-horizon navigation. To address this gap, we propose ReflectVLN, an agentic VLN framework that organizes decision-making through bidirectionally interactive intention and execution agents. The intention agent performs subtask decomposition and reflection, generating executable subtask descriptions as corrective plans. Conditioned on these descriptions, the execution agent grounds them into short-horizon actions under current observations while monitoring sub-goal progress and detecting off-track behavior. Crucially, ReflectVLN enables closed-loop bidirectional communication: the execution agent emits progress and deviation signals to trigger reflection and subtask updates on demand, and the intention agent returns structured guidance that reconditions subsequent actions for recovery. To encourage temporally coherent decisions with interpretable intermediate rationales, we introduce Action Chain-of-Thought (Action-CoT), a path-conditioned dual-query training scheme for action generation. Experiments on standard VLN benchmarks show that ReflectVLN improves success rates and path efficiency under a constrained data budget, with favorable training cost and fewer high-level intention calls at inference time, while providing interpretable intermediate decisions for analysis and collaboration. ## 1. 论文的核心问题与整体含义（研究动机和背景）
- **核心问题**：现有视觉-语言导航（VLN）方法在长时域连续执行中，缺乏显式的闭环机制来跟踪语义进展、诊断执行失败，并从错误累积中恢复。
- **动机与背景**：在连续环境中，小控制误差会随导航步数积累，导致路径偏移。虽然已有层次化或双系统设计（如Hi-Robot、DualVLN），但多为单向通信（慢系统→快系统），缺少执行时的进度/偏差反馈，无法及时触发子目标更新或纠正。此外，缺乏显式的反思机制来检测和修正失败，限制了鲁棒性。

## 2. 论文提出的方法论
- **核心思想**：提出 **ReflectVLN**，一种智能体化框架，包含两个独立参数化的代理：**意图代理（Intention Agent）** 和 **执行代理（Execution Agent）**。两者通过双向交互形成闭环：执行代理不仅输出动作，还发出状态 token 触发意图代理进行反思或子目标更新；意图代理返回结构化指导（子目标或纠正描述），重新调整后续动作。
- **关键技术细节**：
  - **意图代理**：基于 Qwen2.5-VL-3B，输入指令、历史视觉帧和触发 token（`<VLN_BOR>` 正常更新，`<VLN_BOC>` 纠正），生成新子目标文本。
  - **执行代理**：基于 Qwen2.5-VL-3B，输入指令、当前子目标、视觉历史，输出粗路径 `ˆP`、短时域动作序列 `ˆA` 和状态 token：`<VLN_BOA>`（继续执行）、`<VLN_BOR>`（完成子目标）、`<VLN_BOC>`（检测到偏离）。
  - **Action-CoT（动作思维链）**：路径条件双查询架构。先用导航查询预测较长时域的粗路径（`ˆP`），再联合粗路径表示与动作查询预测短时域控制（`ˆA`）。粗路径提供中间监督，增强时序一致性。
  - **两阶段训练**：
    - Stage I：执行代理 Action-CoT 预训练，使用专家轨迹，损失包含位置误差、角度余弦损失和停止损失。

  - **反思数据管线**：一次 rollout 收集失败案例，用专家策略生成纠正轨迹，VLM 生成自然语言描述，再经三部分交叉验证过滤，得到约 100k 反思增强样本并入训练集。

1. **ReflectVLN 在有限数据预算下取得竞争性能**：在 RxR-CE 上 SR 66.0%（优于 DualVLN 61.4%），在 R2R-CE 上 SPL 58.5%（与最佳 DualVLN 持平）。

## 7. 优点
1. **创新的闭环交互机制**：执行代理通过状态 token 主动反馈，意图代理按需反射，实现自发恢复。
2. **Action-CoT 提供中间监督**：粗路径预测作为辅助推理，增强短期控制的全局一致性，且可解释。
3. **数据效率高**：仅使用 R2R+RxR 数据加上一次 rollout 增强（100k 样本），无需大规模预训练或多轮迭代。
4. **训练和推理成本可控**：使用 3B 模型，1.5M 样本训练，推理时降低高级调用次数。
5. **可解释性**：子目标文本输出使导航决策可检视、可分析和可协作。
