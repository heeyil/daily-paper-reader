## 摘要
强化学习（RL）在大语言模型（LLMs）的后训练中正变得越来越重要。先前用于LLMs的RL流程大多采用同步和批交错方式，这对于长周期代理任务效率低下。最近，异步RL作为一种更高效的替代方案出现，它通过随着收集展开数据时更新模型来提高效率。然而，现有的异步RL系统通常强调吞吐量，而训练稳定性和任务有效性在很大程度上尚未被充分探索。例如，一个关键挑战是，广泛使用的GRPO框架中的分组采样并不自然地适用于异步代理训练。在本文中，我们提出了单次展开异步优化（SAO）来解决异步RL中的稳定性和非策略挑战。为了减少非策略效应并提高泛化能力，我们使用单次展开采样替代分组采样，即每个提示仅使用一个展开。我们进一步通过实际的价值模型训练设计改进了这种单次展开策略。为了提高优化稳定性，我们引入了一种严格的双边令牌级裁剪策略。SAO能够稳定训练一千步，并在代理编码和推理基准（如SWE-Bench Verified、BeyondAIME和IMOAnswerBench）上始终优于GRPO及其变体。我们还证明了单次展开RL在模拟在线学习环境中特别有效，其中模型必须适应不断变化的环境。为此，SAO已成功部署在训练开源GLM-5.2模型（750B-A40B）的代理RL流程中。

## Abstract
Reinforcement learning (RL) is becoming increasingly important for post-training large language models (LLMs). Previous RL pipelines for LLMs were mostly synchronous and batch-interleaved, which is inefficient for long-horizon agentic tasks. Recently, asynchronous RL has emerged as a more efficient alternative by updating the model as rollouts arrive. However, existing asynchronous RL systems often emphasize throughput, while leaving training stability and task effectiveness largely underexplored. For example, a key challenge is that group-wise sampling in the widely-used GRPO framework does not naturally fit asynchronous agentic training. In this paper, we present Single-rollout Asynchronous Optimization (SAO) to address the stability and off-policy challenges in asynchronous RL. To reduce off-policy effects and improve generalization, we replace group-wise sampling with single-rollout sampling, that is, using one rollout per prompt. We further improve this single-rollout strategy with practical value-model training designs. To improve optimization stability, we introduce a strict double-side token-level clipping strategy. SAO is able to train stably for one thousand steps and consistently outperform GRPO and its variants on agentic coding and reasoning benchmarks, such as SWE-Bench Verified, BeyondAIME, and IMOAnswerBench. We also demonstrate that single-rollout RL is particularly effective in a simulated online learning setting, where the model must adapt to changing evolving environments. To this end, SAO is successfully deployed in the agentic RL pipeline for training the open GLM-5.2 model (750B-A40B).

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：强化学习（RL）在大语言模型（LLM）后训练中日益重要。然而，现有LLM的RL流程大多采用同步、批交错的模式（如PPO、GRPO），这在长周期代理任务（如代码生成、多回合推理）中效率低下，因为轨迹长度差异大，同步等待导致大量GPU闲置。
- **问题**：异步RL虽然能通过连续消费rollout提高利用率，但引入了两个核心挑战：(1) **训练不稳定性**——异步导致多个版本的旧策略同时存在，产生严重的离策略（off-policy）问题；(2) **GRPO框架不兼容**——GRPO依赖于同一提示的分组采样（group-wise sampling）进行优势估计，这在异步场景下必须等待最慢的组内样本，且不适用于只有单条轨迹反馈的在线/复杂环境。
- **整体含义**：本文旨在设计一种既能保持异步效率，又能稳定训练并提升任务有效性的新异步RL算法。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- **单次展开（Single-rollout）**：放弃GRPO的分组采样，每个提示仅用一个rollout立即进行训练更新，从而避免同步等待和离策略累积。
- **双面令牌级重要性采样与裁剪（Double-sided Importance Sampling, DIS）**：直接用`π_rollout`作为行为策略，`π_θ`作为当前策略，计算比率`r_t(θ)=exp(log π_θ(at|st) - log π_rollout(at|st))`；并采用严格的双侧裁剪，将`r_t(θ)`限制在`[1-ε_l, 1+ε_h]`内，超出该区间的令牌梯度置零（masking），以稳定训练。
- **增强值模型训练**：为降低单次展开的高方差，提出三种技术：
  - **更快值更新（Faster Value Update）**：每个策略梯度步对值网络进行K次更新（K=2），使价值估计更快适应策略变化。
  - **冻结注意力参数（Frozen-Attention）**：值模型训练时冻结全注意力层，仅优化MoE层，避免梯度爆炸和训练不稳定。
  - **跳过观察令牌的GAE（Skip-Observation Token-level GAE）**：针对代理任务中`动作-环境反馈-动作`的非连续轨迹，跨过环境反馈令牌计算优势，只依赖模型本身的输出。
- **扩展值预训练数据**：增加值模型预训练语料规模，解决“冷启动”问题。

### 关键技术细节（公式化说明）
- 优化目标：`L(θ) = E_t[ f(r_t(θ), ε_l, ε_h) * A_t * log π_θ(at|st) ]`，其中`f`为校准函数，当`1-ε_l < r_t < 1+ε_h`时取`r_t`，否则取0。
- 优势估计：采用Skip-Observation GAE，`A(a_i,N) = δ + γλ * A(a_{i+1,0})`，其中`δ = r_t + γV(a_{i+1,0}) - V(a_i,N)`。
