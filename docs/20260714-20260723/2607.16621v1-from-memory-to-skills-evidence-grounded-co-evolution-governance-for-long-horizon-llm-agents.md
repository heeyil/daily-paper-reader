## 摘要
现有的长周期LLM智能体记忆系统通常将先前的痕迹作为被动上下文检索，而不是将其转化为可执行的能力。在本文中，我们提出了MSCE，一种无需训练的记忆-技能协同演化框架，它将智能体经验组织为有依据的步骤痕迹、可重用的过程策略和声明式环境认知。MSCE将具有正面估计收益的、有证据支持的L2策略结晶为可调用的技能，这些技能保留了证据链接、适用范围边界、决策指导、验证规则和可靠性估计。它进一步引入了反思加权值回填，通过密集的局部自我反思传播稀疏的终端反馈，以产生证据校准的痕迹值，用于管理记忆和技能演化。在EvoAgentBench和LoCoMo上的实验表明，MSCE显著优于最先进的技能增强和记忆驱动智能体基线，表现出强大的跨域迁移能力和终身演化能力。

## Abstract
Existing memory systems for long-horizon LLM agents often retrieve prior traces as passive context rather than converting them into executable capabilities. In this paper, we propose MSCE, a training-free Memory--Skill Co-Evolution framework that organizes agent experience into grounded step traces, reusable procedural policies, and declarative environmental cognition. MSCE crystallizes evidence-backed L2 policies with positive estimated gain into callable skills that retain evidence links, applicability boundaries, decision guidance, verification rules, and reliability estimates. It further introduces reflection-weighted value backfilling, which propagates sparse terminal feedback through dense local self-reflections to produce evidence-calibrated trace values for governing memory and skill evolution. Experiments on EvoAgentBench and LoCoMo demonstrate that MSCE significantly outperforms state-of-the-art skill-augmented and memory-driven agent baselines, exhibiting strong cross-domain transferability and lifelong-evolution capabilities.

- **整体含义**：论文旨在解决记忆系统与技能系统之间的鸿沟，提出一个**无需训练的、记忆–技能共演化框架（MSCE）**，使智能体能够从经验中自主提取可复用的过程性知识，并在持续交互中动态维护和升级。
- **背景意义**：随着 LLM 智能体在长期、多步、用户特定任务中的部署（如工具使用、文件编辑、应用交互），长期记忆对于突破上下文窗口限制、保持一致性、个性化行为至关重要。现有方法（事实记忆/经验记忆）多停留在被动检索，缺乏向操作知识的转化。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
- 将代理经验组织为**三级记忆层次**：
  - **L1 痕迹记忆（Trace Memory）**：存储有依据的步骤级证据（状态、动作、观察、自我反思、价值）。
  - **L2 策略记忆（Policy Memory）**：从跨回合痕迹中归纳可复用的过程性策略（触发条件、自然语言过程、验证规则、适用范围边界、支持证据集）。
  - **L3 环境认知记忆（Environmental Cognition Memory）**：抽象关于环境结构与约束的声明性知识（实体/结构事实、行动–响应规律、约束条件、支持策略集）。
- 在三级记忆基础上，将**有证据支持、正面收益估计、稳定一致的 L2 策略结晶为可调用技能**，技能保留证据链接、决策指导、可靠性估计等字段。
- 引入**反思加权值回填（Reflection-Weighted Value Backfilling）**，耦合稀疏的终端反馈与密集的局部自我反思，生成证据校准的痕迹价值，统一驱动记忆检索、策略归纳、环境认知抽象、技能晋升与修订。

### 关键技术细节
- **L1 痕迹价值更新公式**（式 2）：
  \[
  V(f_{i,t}^{(1)}) = \alpha_{i,t} R_i + (1 - \alpha_{i,t}) \gamma V(f_{i,t+1}^{(1)})
  \]
  其中 \(\gamma \in [0,1)\) 为折扣因子，\(\alpha_{i,t} \in [0,1]\) 为反思权重，由 LLM 评估自我反思的忠实性、因果洞察、可迁移性和具体性后得到。
- **L2 策略收益计算**（式 1）：
  \[
  G(f^{(2)}) = \bar{V}_{\text{with}} - \bar{V}_{\text{blend}}(S_{\text{without}})
  \]
  通过对比关联该策略的痕迹与未关联痕迹的平均价值，判断策略效用。
- **技能结晶**：
  - 条件：源 L2 策略保留支持证据、满足正收益阈值 \(G > \theta_G\)、且近期证据稳定。
  - 结晶过程由 LLM 提示驱动，生成结构化字段（触发条件、过程、验证、边界、证据锚点、决策指导、可靠性），并通过确定性验证器检查格式、工具白名单、证据可追溯性。
- **技能可靠性估计**：平滑成功率 \(\eta = (n_{\text{pass}} + 1) / (n_{\text{trial}} + 2)\)，用于生命周期管理（试用→活跃→归档）。

### 算法流程（文字说明）
1. **交互阶段**：智能体每一步生成痕迹（状态、动作、观察、自我反思）；触发技能条件时注入技能上下文。
2. **终端反馈到达**：接收环境奖励或用户文本反馈（通过 LLM 量化成标量 \(R_i\)）。
3. **价值回填**：使用反思权重将 \(R_i\) 沿时间步反向传播，更新所有 L1 痕迹价值。
4. **记忆更新**：
   - 高价值痕迹尝试关联或归纳 L2 策略（需跨回合证据）。
   - 活跃 L2 策略可能触发 L3 环境认知抽象。
   - 满足条件的 L2 策略结晶为技能（经验证后以试用状态部署）。
5. **推理阶段**：按层次检索（技能 → 环境认知 → 痕迹），优先使用活跃技能，失败回退。
