## 摘要
世界模型为长程LLM智能体提供了一种原则性的前瞻能力：在执行前预测行动后果。然而，不可靠的前瞻可能被忽视、误用，甚至降低下游决策质量。本文提出WorldEvolver，一种自我演化的世界模型框架，其在保持下游智能体和所有模型参数冻结的同时，修正部署时的上下文。WorldEvolver集成三个模块：(i) 情景记忆，通过基于检索的模拟利用真实行动转移；(ii) 语义记忆，从预测-观察不匹配中提取持久启发式规则；(iii) 选择性前瞻，在将低置信度预测融入智能体推理上下文之前进行过滤。我们在ALFWorld和ScienceWorld上评估WorldEvolver，在Word2World上衡量世界模型预测准确性，在AgentBoard上衡量下游智能体成功率。大量实验表明，WorldEvolver在三个主干网络上取得最高预测准确率，并在下游智能体成功率上领先其他世界模型基线，证明测试时记忆修正能同时提升预测保真度和规划性能。

## Abstract
World models offer a principled way to equip long-horizon LLM agents with foresight: predictions of action consequences before execution. However, unreliable foresight can be ignored, misused, or even degrade downstream decision-making. In this paper, we introduce WorldEvolver, a self-evolving world model framework that revises its deployment-time context while keeping the downstream agent and all model parameters frozen. WorldEvolver integrates three modules: (i) Episodic Memory, which exploits real action transitions through retrieval-based simulation; (ii) Semantic Memory, which extracts persistent heuristic rules from prediction-observation mismatches; and (iii) Selective Foresight, which filters low-confidence predictions before integrating them into agent reasoning context. We evaluate WorldEvolver on ALFWorld and ScienceWorld, measuring world model prediction accuracy on Word2World and downstream agent success rate on AgentBoard. Extensive experiments show that WorldEvolver achieves the highest prediction accuracy across three backbones and leads other world model baselines on downstream agent success rate, demonstrating that test-time memory revision enhances both predictive fidelity and planning performance.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：LLM智能体在长程任务中依赖世界模型进行前瞻预测（foresight），但部署时面临环境分布偏移，导致模型预测不准确，进而可能误导智能体决策。
- **现有方法局限**：冻结的世界模型无法适应新分布；离线微调或参数更新计算成本高，且易导致灾难性遗忘；部分方法（如RAWM-ϕ、ITP-I）的前瞻预测不可靠时反而降低智能体性能（论文图2的预实验证实了“噪声前瞻”会损害行动准确率）。
- **核心动机**：需要一种能在测试时自我演化（self-evolving）的世界模型，在不更新模型参数的前提下，通过修正外部记忆来改善预测可靠性，从而提升下游智能体的规划能力。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：WorldEvolver是一个**免训练、自我演化**的世界模型框架。它保持智能体策略和世界模型的所有参数冻结，仅在部署时修正世界模型赖以推理的**外部记忆上下文**（即prompt-level evidence）。
- **三大模块**：
  - **情景记忆（Episodic Memory, ME）**：存储已执行的真实转移三元组 (oᵢ, aᵢ, oᵢ₊₁)。在预测时，根据当前候选动作通过**Jaccard相似度**检索前 kME 个最相似的历史转移，作为检索式模拟的上下文。
  - **语义记忆（Semantic Memory, MS）**：将预测-观察不匹配（mismatch）转化为持久化的启发式规则。通过**因子化状态（factorized tuples）比较**判断是否出现不匹配，然后由LLM批评器生成规则（如“检查物体不会改变它位置”），每条规则附带证据分数 eᵢ，仅保留 eᵢ > 0 的规则。语义记忆支持批量更新（kMS 个不匹配后合并更新）。
  - **选择性前瞻（Selective Foresight）**：基于生成预测的**几何平均token概率**计算置信度 qₜ，当 qₜ ≥ τ 时才将预测传递给智能体；否则不提供任何前瞻。这避免了低可靠预测误导智能体（见预实验图2）。
- **算法流程（Algorithm 1）**：
  1. 智能体生成草稿动作 a⁽⁰⁾ₜ。
  2. 从 ME 检索 kME 个历史转移。
  3. 世界模型在 (sₜ, a⁽⁰⁾ₜ, ME, MS) 下输出预测 ôₜ₊₁ 及置信度 qₜ。
  4. 选择性前瞻：若 qₜ ≥ τ 则 fₜ = ôₜ₊₁，否则为空。
  5. 智能体基于 (sₜ, fₜ) 采样执行动作 aₜ = a⁽¹⁾ₜ。
  6. 若最终动作与草稿不同，重新查询世界模型获取对齐的预测。
  7. 执行动作后，将真实转移追加到 ME；比较预测与真实观测，若因子化后不一致则通过LLM批评器累积更新 MS。

## 7. 优点

- **免训练**：所有模型参数冻结，仅通过外部记忆上下文演化，避免了参数更新的计算开销和遗忘问题。
- **模块化可解释**：情景记忆提供具体案例，语义记忆提供人类可读的启发式规则，选择性前瞻明确给出了置信度门控，每一步都可以审计。
- **跨骨干/策略的鲁棒性**：在两种不同推理能力的智能体（ReAct, ReflAct）和多种规模的大模型上均取得一致改进。
- **实用性**：运行时开销合理（每步 1~2 秒），适合在线部署。

## 8. 不足与局限

- **环境覆盖局限**：仅测试于两类文本交互环境（家务模拟和科学实验），缺乏对网页导航、代码生成、多模态或真实机器人环境的验证，泛化性未定。
