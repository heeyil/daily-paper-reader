## 摘要
我们提出Agents-A1，一个35B参数的混合专家智能体模型，通过扩展智能体任务视野达到万亿参数级别的性能。我们从两个角度研究智能体任务视野的扩展：扩展长程轨迹和扩展异构智能体能力。为支持此目标，我们构建了一个长程知识-动作基础设施，连接外部知识、动作、观察和验证器结果，产生平均长度为45K标记的智能体轨迹。基于此，我们采用三阶段流程训练Agents-A1。首先，进行全领域监督微调，使基础模型对齐广泛的智能体行为。其次，训练领域级教师模型以捕获每个领域的专业知识。第三，我们提出一种多教师领域路由的在线策略蒸馏方法，结合显著词汇对齐，以提高跨不同领域的知识迁移效率，将六个异构领域统一到一个可部署的学生模型中。Agents-A1在长程智能体基准测试中表现出强大且广泛的性能。与1T参数模型如Kimi-K2.6和DeepSeek-V4-pro相比，Agents-A1在SEAL-0（56.4）、IFBench（80.6）、HiPhO（46.4）、FrontierScience-Olympiad（79.0）和MolBench-Bind（56.8）上取得了领先结果，并在SciCode（44.3）、HLE（47.6）和BrowseComp（75.5）上保持高度竞争力。我们希望这项工作为社区提供一条实用路径，即使用35B智能体扩展任务视野，从而在长程任务上达到或匹配1T模型的性能。

## Abstract
We introduce Agents-A1, a 35B Mixture-of-Experts Agentic Model that reaches trillion-parameter-level performance by scaling the agent horizon. We investigate agent-horizon scaling from two perspectives: scaling long-horizon trajectories and scaling heterogeneous agent abilities. To support this goal, we build a long-horizon knowledge-action infrastructure that connects external knowledge, actions, observations, and verifier outcomes, producing agentic trajectories with an average length of 45K tokens. Based on this, we train Agents-A1 with a three-stage recipe. First, we perform full-domain supervised fine-tuning to align the base model with broad agentic behaviors. Second, we train domain-level teacher models to capture specialized expertise in each domain. Third, we propose a multi-teacher domain-routed on-policy distillation with salient vocabulary alignment to improve knowledge transfer efficiency across different domains, unifying six heterogeneous domains into one deployable student model. Agents-A1 achieves strong and broad performance for long-horizon agent benchmarks. Compared with 1T-parameter model such as Kimi-K2.6 and DeepSeek-V4-pro, Agents-A1 achieves leading results on SEAL-0 (56.4), IFBench (80.6), HiPhO (46.4), FrontierScience-Olympiad (79.0), and MolBench-Bind (56.8), and remains highly competitive on SciCode (44.3), HLE (47.6) and BrowseComp (75.5). We hope this work provides the community with a practical path for scaling the horizon using a 35B agent that can reach or match the performance of 1T models on long-horizon tasks.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- **通过扩展智能体视野而非参数规模**来提升长程任务性能。具体从两个维度切入：① 扩展长程轨迹（long-horizon trajectories）；② 扩展异构智能体能力（heterogeneous agent abilities）。

### 关键技术细节

#### (1) 长程知识-动作基础设施（Long-Horizon Knowledge-Action Infrastructure）
- 构建**知识-动作图（KAG）**，将外部知识、动作、观察和验证器结果作为一等对象链接起来。每个域 \( B_d \) 的 KAG 定义为四元组 \( G_d = (C_d, A_d, O_d, V_d) \)，其中：
  - \( C_d \)：域语料库（证据块、实体、事实等）
  - \( A_d \)：动作空间（工具调用、检索查询、代码编辑等）
  - \( O_d \)：观察空间（工具返回、检索证据、执行状态）
  - \( V_d \)：验证器集合（正确性、证据支持等检查）
- 通过提议者-求解者-验证者自博弈循环（proposer–solver–verifier game）扩展 KAG，生成约束充分、过程可验证、证据覆盖的轨迹，平均长度 **45K token**。

#### (2) 三阶段训练流程
1. **全领域监督微调（Full-domain SFT）**：在约10万条平均45K token的轨迹上微调 Qwen3.5-35B-A3B，覆盖深度搜索、代码工程、科学推理、指令跟随、通用智能体任务等多个领域。
2. **域级教师模型训练（Domain-level Teacher Training）**：
   - **搜索领域**：SFT + GRPO强化学习，使用搜索、读页、代码工具，奖励包含正确性、搜索效率惩罚、格式校准。
   - **科学推理**：两阶段SFT（先纯推理增强，后工具增强）。
   - **指令跟随**：两阶段RL（先约束跟随，后长上下文学习）。
   - **工具调用**：SFT + RL，使用过程奖励和不对称优势（PAPO）。
3. **多教师域路由在线蒸馏（Multi-teacher Domain-routed On-policy Distillation with Salient Vocabulary Alignment）**：
   - 学生模型基于自身展开（on-policy rollout）接受对应域教师的指导。
   - **显著词汇对齐（Salient Vocabulary Alignment, SVA）**：在教师选择的Top-k token集上计算截断反向KL散度，而非全词汇。
   - **域归一化目标**：在每个活跃域内平均损失再跨域平均，防止高频域主导更新。

### 公式说明（文字描述）
- SVA损失：每个训练位置 \(t\)，在教师分布 \(p_{t,i}\) 的Top-k支撑集 \(S^{(k)}_{i,t}\) 上，重新归一化学生分布 \(\bar{p}_{s'}\) 和教师分布 \(\bar{p}_{t,i}\)，计算反向KL散度并平均。
- 域归一化损失：\( L = \frac{1}{|D_B|} \sum_{d \in D_B} \frac{1}{|B_d|} \sum_{i \in B_d} \ell_{\text{SVA}}^{(i)} \)，其中 \(D_B\) 是活跃域集合。
